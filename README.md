Chatty App
=====================

A live chat application built using ReactJS.



### Usage

Install the dependencies and start the server.

```
npm install
npm start
open http://localhost:3000
```

### Dependencies

* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react
* css-loader
* node-sass
* sass-loader
* sockjs-client
* style-loader
* webpack
* webpack-dev-server
* react
* react-dom
* uuid
